# Algorithm Performance Comparison

This repository contains exercises measuring and comparing the execution time of different algorithm implementations.

## Exercises Overview

1. **Baseline Execution Measurement**  
   - Execute a program multiple times to measure execution time.
   - Record times and compute the average.

2. **Algorithm Comparison**  
   - Run two implementations (Naive vs Single-Pass) at least three times each.
   - Record execution times and results.
   - Compute the average execution time for each.
   - Compute the relative difference (how many times one is slower than the other).

> The goal is to practice performance measurement, computation of averages, and comparison of algorithm efficiency.
